# AI-Bridge (Side Panel Edition)

閲覧中のウェブページの「テキスト情報」と「スクリーンショット」を瞬時に取得し、サイドパネルに表示。お好みのAIチャット（Gemini, ChatGPT, Claude, etc...）へシームレスに橋渡しするChrome拡張機能です。

## 特徴

- **ダイナミック・サイドパネル・ヘッダー**: サイドパネル上部のヘッダーから直接AIサービスを切り替え可能。
- **マルチAI対応**: Gemini, ChatGPT, Claude, Grok, DeepSeek, Qwen など、主要なAIを自在に切り替え。
- **プレミアムUI**: ガラスモーフィズムを採用したモダンなオプションページ。快適な操作性を提供します。
- **スマート・テーマ管理**: 
    - OSの設定に連動する「自動モード」に加え、手動での「ライト/ダーク」固定が可能。
    - 設定変更はサイドパネルへ即座に反映されます。
- **一括コピー性能**: 右クリックメニューやアイコンクリック一つで、以下の情報をクリップボードに同時格納：
    - ページのタイトルとURL
    - 選択範囲のテキスト（選択時のみ）
    - 抽出されたコンテキスト（HTML / Markdown / 本文のみ）
    - スクリーンショット画像（可視範囲 / ページ全体）
- **強力なテキスト抽出**:
    - **マークダウン形式**: 不要なHTMLタグを削ぎ落とした読みやすい形式。
    - **本文のみ抽出 (Smart)**: 記事のメインコンテンツのみをAIが自動判別。トークン消費を最小限に。
- **設定のオートセーブ**: AIリスト以外の設定項目は、変更した瞬間に自動保存。保存ボタンを何度も押す必要はありません。

## インストール方法

1. このリポジトリをダウンロードし、展開します。
2. Chromeブラウザで `chrome://extensions/` を開きます。
3. 右上の「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」を選択し、本フォルダを選択します。

## 使い方

1. 任意のウェブページを開きます。
2. ページ上で右クリックし「**AIにこのページ情報をコピーして開く**」を選択します。
3. サイドパネルが開きます。AIの入力欄で `Ctrl + V` を行うと、テキストと画像が一度にセットされます。
4. サイドパネル上部のドロップダウンで、瞬時に別のAIへ切り替えて比較も可能です。

## 設定

オプションページ（アイコン右クリック > オプション）で詳細なカスタマイズが可能です：
- **デフォルトAI**: 起動時に表示するAI。
- **外観テーマ**: 自動（システム連動） / ライト / ダーク。
- **カスタムAIリスト**: 独自のURLを持つAIサービスを追加編集。
- **抽出モード/撮影範囲**: 用途に応じた最適なデータ取得方法を選択。

## 免責事項

- 本拡張機能は情報のコピーを補助するツールです。AIへの自動的なプロンプト送信は行いません（安全のため、ユーザー自身がペーストすることを前提としています）。
- 一部のサイトでは、セキュリティ上の理由（CSP等）によりiframe内での表示が制限される場合があります。
