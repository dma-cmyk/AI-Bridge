# AI-Bridge (Side Panel Edition)

閲覧中のウェブページの「テキスト情報」と「スクリーンショット」を瞬時に取得し、サイドパネルに表示したお好みのAIチャット（Gemini, ChatGPT, Claude, etc...）へ簡単に渡すことができるChrome拡張機能です。

## 特徴

- **マルチAI対応**: Gemini, ChatGPT, Claude, Grok, DeepSeek, Qwen など、主要なAIチャットをサイドパネルで切り替えて利用可能。
- **カスタムAIリスト**: 使いたいAIのURLを自由に追加・編集できます。
- **一括コピー機能**: 右クリックメニューやアイコンクリック一つで、以下の情報をクリップボードに同時格納します：
  - ページのタイトルとURL
  - 選択範囲のテキスト（選択時のみ）
  - ページのソースまたは本文テキスト
  - 表示範囲またはページ全体のスクリーンショット
- **強力なテキスト圧縮**:
  - **HTMLソースコード**: ページの構造をそのままコピー。
  - **マークダウン形式**: タグを取り除いたクリーンなテキスト。
  - **本文のみ抽出**: 記事のメインコンテンツだけを自動判別して抽出（トークンを劇的に節約）。
- **ページ全体キャプチャ**: 自動スクロールによる全体スクリーンショットの撮影に対応。
- **DNRによるFrame制限解除**: iframeでの表示を制限しているサイト（Gemini等）もサイドパネル内で表示できるよう、特殊なルールで制限を解除しています。

## インストール方法

1. このリポジトリをクローンまたはダウンロードします。
2. Chromeブラウザで `chrome://extensions/` を開きます。
3. 右上の「デベロッパーモード」をオンにします。
4. 「パッケージ化されていない拡張機能を読み込む」を選択し、このリポジトリのフォルダを選択します。

## 使い方

1. 任意のウェブページを開きます。
2. ページ上で右クリックし「**AIにこのページ情報をコピーして開く**」を選択するか、拡張機能アイコンをクリックします。
3. サイドパネルが開きます。AIの入力欄で `Ctrl + V`（貼り付け）を行うと、テキストと画像が一度にセットされます。

## 設定

拡張機能のオプション画面から以下の設定が可能です：
- デフォルトで開くAIサービスの選択
- カスタムAIリストの編集（名前,URL形式）
- データ抽出形式の選択（HTML / Markdown / 本文のみ）
- スクリーンショットの撮影範囲（表示範囲 / ページ全体）

## 免責事項

- 本拡張機能はクリップボードを介した情報の受け渡しのみを行います。AIへの自動的な入力送信機能は、各サイトの利用規約やセキュリティを考慮し、意図的に実装していません。
- サイトによっては、iframe内での表示が完全に制限されている場合があります。
