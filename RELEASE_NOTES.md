# リリースノート - AI-Bridge

## v1.0.3 (2026-03-14)
- **右クリックメニューの挙動修正**: 
  - 右クリックメニュー（コンテキストメニュー）から起動した際は、自動キャプチャを行わずサイドパネル（チャット画面）を開くだけにしました。
  - アイコンクリック時は引き続き自動キャプチャとコピーが実行されます。
- **メニュータイトルの変更**: 文言を「AIとのチャットをサイドパネルで開く」に更新しました。

## v1.0.2 (2026-03-14)
- **オプションページのデザイン刷新**: グラスモーフィズム、モダンなカラーパレット、アニメーションを採用し、プレミアムな外観にアップデートしました。
- **UI/UXの改善**: 設定項目をカード形式に整理し、現在の選択状態を視覚的に分かりやすくしました。

## v1.0.1 (2026-03-14)
- **拡張機能アイコンの追加**: 16x16, 48x48, 128x128 のアイコン画像を追加しました。

## v1.0.0 (2026-03-14)
- **初期リリース**: 
  - ページのソースコードとスクリーンショットを取得し、サイドパネルのGeminiへ転送する機能を実装しました。
