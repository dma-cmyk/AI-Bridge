# リリースノート - AI-Bridge
 
+## v1.2.0 (2026-03-15)
+- **マイク（音声入力）への対応**:
+  - サイドパネル内のAIチャット（Gemini, ChatGPT等）でマイクが使用可能になりました。
+  - 拡張機能としてのセキュリティ制限を考慮し、ブラウザ標準の権限管理とスムーズに連携する設計を採用しました。
+- **「プライバシーと権限」設定の追加**:
+  - オプションページに、マイク権限などを一括管理できるセクションを新設しました。
+  - Chromeの「サイト詳細設定」をワンクリックで開くボタンを配置し、権限の許可やブロック解除が誰でも簡単に行えるようになりました。
+- **UIのクリーンアップ**:
+  - サイドパネルから不要なボタンを排除し、コンテンツに集中できるシンプルなデザインを維持しました。
+
 ## v1.1.0 (2026-03-15)
- - **サイドパネル・ヘッダーの実装**:
-   - サイドパネル上部から直接AIサービスを切り替えられるドロップダウンを追加しました。
-   - ヘッダー内のギアアイコンから即座にオプションページへアクセス可能になりました。
- - **設定のオートセーブ導入**:
-   - オプションページでの設定（AIサービス、テーマ、抽出形式、キャプチャ範囲）が、変更時に自動保存・即時反映されるようになりました。
-   - 「カスタムAIリスト」のみ、専用の保存ボタンで明示的に更新する安全な設計に変更しました。
- - **テーマ制御の安定化**:
-   - オプションページでの「自動/ライト/ダーク」設定が、サイドパネルのヘッダー色へより確実に同期されるよう内部ロジックを整理しました。
-   - ページごとの不安定な自動検知を廃止し、ユーザーの設定通りに表示される安定性を優先しました。
- 
- ## v1.0.5 (2026-03-14)
- - **「画像のみ取得」機能の追加**:

-   - このモードでは、クリップボードにスクリーンショット画像のみをコピーし、Markdown等のテキスト説明を省略します。
